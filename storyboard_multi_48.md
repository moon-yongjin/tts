# 🎬 대본 기반 [다인 연출] 48장 스토리보드 가이드

기본 은숙 단독 컷 위주에서 벗어나, 대본의 상황에 맞게 **조연(전남편, 시어머니, 할머니 등)이 함께 등장**하며, 연사 속도 가속을 위해 **해상도를 20% 축소(576x1024)**한 16개 씬, 총 48장 피드백 배정 가이드입니다.

---

## 📸 장면 및 다중 액터 매칭 (Scene 1~16)

| 씬 번호 | 등장 인물 연출 (다중 인원) | 파일명 접두사 (Var1~3) |
| :--- | :--- | :--- |
| **Scene 01** | **은숙 단독** - 새벽 5시 주방에서 시어머니 약 챙기기, 도시락 싸기 (피로/처량) | `01_Mornings_Hardwork` |
| **Scene 02** | **은숙 + 시어머니** - 침실에서 치매 시어머니 부양하기 (고단함) | `02_Caring_Grandmother` |
| **Scene 03** | **은숙 + 전남편(정태)** - 식탁에 마주 앉아 남편의 이혼 통보를 받는 충격적인 씬 | `03_With_Cold_Husband` |
| **Scene 04** | **은숙 단독** - 법률 오피스 미팅실에서 위자료 3천 합의서에 도장 찍는 슬픈 모습 | `04_Signing_Papers` |
| **Scene 05** | **은숙 단독** - 이혼 이면의 배신(정태-미영)을 깨닫고 울음을 터뜨림 (절망) | `05_Shock_Truth` |
| **Scene 06** | **은숙 단독** - 8평 원룸 골방에 혼자 멍하니 앉아 있는 고독한 모습 | `06_Isolated_Studio` |
| **Scene 07** | **은숙 단독** - 가계부의 숫자들을 돋보기/안경 쓰고 꿰뚫어 보며 집중하는 모습 | `07_Ledger_Accounting` |
| **Scene 08** | **은숙 + 옆집할머니** - 문을 두드리고 깍두기를 주며 뜻밖의 희망을 주는 순간 | `08_Granny_Visiting` |
| **Scene 09** | **은숙 단독** - 재래시장 노점에 '하는숙 반찬' 간판을 직접 다는 희망찬 모습 | `09_Store_Opening` |
| **Scene 10** | **은숙 + 손님** - 시장 부스에서 다채로운 반찬팩 위에 손을 얹고 대박 미소를 지음 | `10_Success_Moment` |
| **Scene 11** | **은숙 + 아들** - 사죄하는 아들의 손을 맞잡고 함께 눈물짓는 모자 상봉 씬 | `11_Son_Apology` |
| **Scene 12** | **은숙 + 현우(새사랑)** - 책방 주인 현우 씨가 시장에서 말없이 무거운 반찬통 들어줌 | `12_Bookstore_Helper` |
| **Scene 13** | **은숙 + 현우** - 테이블 주위에서 따뜻하게 차를 마시며 위로받는 무드 | `13_Touched_Feelings` |
| **Scene 14** | **은숙 단독** - 비밀 서류, 정교한 회계 장부들을 Desk에서 분석하는 지적 단호함 | `14_Evidence_Revenge` |
| **Scene 15** | **은숙 단독** - 등기부를 쥐고 당당하게 승리를 거두는 커리어 스탠스 | `15_Victorious_Ruin` |
| **Scene 16** | **은숙 + 현우** - 해피엔딩. 책방 골목 노을 길에서 현우 씨와 손을 잡은 밝은 웃음 | `16_Happy_Ending` |

---

> [!IMPORTANT]
> *   **가속화 규격**: 사용자 피드백을 반영해 기존 720x1280에서 **576x1024**로 20% 축소(고정 비율)하여 출력 속도가 극대화되었습니다.
> *   **다인 지원**: 은숙 위주의 Portrait에서 벗어나 `pose_desc` 내에서 할머니, 아들, 전남편과의 투샷 연기를 유연하게 허용하도록 설계해 스토리 무드가 선명해졌습니다.
