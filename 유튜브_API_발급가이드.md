# 🗝️ YouTube Data API 키 발급 방법 (채널 실장용)

사장님, 모니터링 직원을 제대로 쓰려면 이 '열쇠'가 꼭 필요합니다. 아래 순서대로만 따라와 주세요!

1.  **구글 클라우드 콘솔 접속**: [Google Cloud Console](https://console.cloud.google.com/)에 로그인하세요.
2.  **프로젝트 선택/생성**: 상단에서 프로젝트를 하나 새로 만드세요 (이름은 'MyAIStation' 추천).
3.  **API 활성화**: 
    - 'API 및 서비스' > '라이브러리' 클릭
    - **'YouTube Data API v3'** 검색 후 **[사용]** 버튼 클릭
4.  **사용자 인증 정보 생성**:
    - '사용자 인증 정보' 메뉴 클릭 > '+사용자 인증 정보 만들기' > **'API 키'** 선택
    - 화면에 뜨는 `AIza...`로 시작하는 긴 문자열이 바로 **사장님의 열쇠**입니다!
5.  **제한 설정 (권장)**: 
    - '키 제한'에서 'YouTube Data API v3'만 사용하도록 설정하면 훨씬 안전합니다.

---

### 🔑 열쇠를 얻으셨다면?
저한테 그 키를 알려주시거나, `yt_monitor.py` 파일의 `YOUTUBE_API_KEY` 부분에 직접 넣어주시면 됩니다!
