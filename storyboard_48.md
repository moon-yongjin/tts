# 🎬 대본 기반 48장 스토리보드 가이드

수정하신 단축 대본 21줄의 서사 흐름에 맞춘 **16개 씬, 총 48장**의 스토리보드 연쇄 생성 현황 가이드입니다. 각 씬별로 3가지의 각도/표정 배리이이션이 생성됩니다.

---

## 📸 장면 노드 매칭 (Scene 1~16)

| 씬 번호 | 대본 주요 내용 및 연출 컨셉 | 파일명 접두사 (Var1~3) |
| :--- | :--- | :--- |
| **Scene 01** | 새벽 5시 주방에서 시어머니 약 챙기기, 도시락 싸기 (피로/처량) | `01_Mornings_Hardwork` |
| **Scene 02** | 침실에서 치매 시어머니 돌보기/부양하기 (고단함) | `02_Caring_Grandmother` |
| **Scene 03** | 식탁에 앉아 남편(정태)의 이혼 통보를 받는 순간 (충격/절망) | `03_With_Cold_Husband` |
| **Scene 4** | 변호사 미팅실에서 위자료 3천 합의서에 도장 찍는 슬픈 모습 | `04_Signing_Papers` |
| **Scene 05** | 이혼 이면의 배신(정태-미영)을 깨닫고 울음을 터뜨림 (절망) | `05_Shock_Truth` |
| **Scene 06** | 8평 원룸 골방에 혼자 멍하니 앉아 있는 고독한 모습 | `06_Isolated_Studio` |
| **Scene 07** | 가계부의 숫자들을 돋보기/안경 쓰고 꿰뚫어 보며 집중하는 모습 | `07_Ledger_Accounting` |
| **Scene 08** | 옆집 할머니가 문을 두드리고 깍두기를 요청하는 순간 (뜻밖의 희망) | `08_Granny_Visiting` |
| **Scene 09** | 재래시장 노점에 '하는숙 반찬' 나무 간판을 직접 다는 모습 | `09_Store_Opening` |
| **Scene 10** | 시장 부스에서 다채로운 반찬팩 위에 손을 얹고 대박 미소를 지음 | `10_Success_Moment` |
| **Scene 11** | 아들이 사죄하며 손을 맞잡고 함께 눈물짓는 모자 상봉 씬 | `11_Son_Apology` |
| **Scene 12** | 책방 주인 현우 씨가 시장에서 무거운 반찬통을 말없이 들어줌 | `12_Bookstore_Helper` |
| **Scene 13** | 북카페 서재 안에서 따뜻하게 차를 마시며 위로받는 무드 | `13_Touched_Feelings` |
| **Scene 14** | 비밀 서류, 정교한 회계 장부들을 Desk에서 분석하는 지적인 단호함 | `14_Evidence_Revenge` |
| **Scene 15** | 오피스텔 등기부를 쥐고 당당하게 승리를 거두는 커리어 스탠스 | `15_Victorious_Ruin` |
| **Scene 16** | 해피엔딩. 책방 골목 노을 길에서 현우 씨와 손을 잡은 밝은 웃음 | `16_Happy_Ending` |

---

> [!TIP]
> *   모든 이미지의 은숙은 **피트니스/모델 가중치 없이 일반적인 주부상(40대 후반의 동안)**으로 고정되어 출하됩니다.
> *   각 장면당 **숫자 1, 2, 3은 각도 및 거리 배리에이션(클로즈업 등)**을 나타내므로 쇼츠 비율 세팅 시 더 다채로운 화면 컷 편집이 가능합니다.
