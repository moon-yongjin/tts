# 🎙️ 1-3 제로샷 스튜디오 (ZeroShot Studio)

제로샷(Zero-Shot) 관련 모든 스크립트와 워크플로우를 체계적으로 관리하기 위한 전용 작업 폴더입니다.

## 📂 폴더 구조

- **`scripts/`**: 모든 파이썬(.py) 실행 코드가 보관되어 있습니다.
- **`outputs/`**: 제로샷 생성 결과물(WAV, MP3, SRT 등)이 저장되는 전용 공간입니다.
- **`configs/`**: 프리셋이나 설정 파일들이 위치합니다.
- **루트 폴더**: 자주 사용하는 실행 파일(.command)들이 위치하여 바로 실행할 수 있습니다.

## 🚀 사용 방법

1. `1-3_ZeroShot_Studio` 폴더를 엽니다.
2. 원하는 제로샷 목소리나 워크플로우 이름의 `.command` 파일을 더블 클릭하여 실행합니다.
3. 생성된 결과물은 `outputs/` 폴더 또는 스크립트에 지정된 다운로드 경로에서 확인하세요.

## 🔧 주요 파일
- **`1-3-92_ZeroShot_UnifiedWorkflow.py`**: 통합 제로샷 생성 마스터 스크립트.
- **`1-3-93_ZeroShot_DualSpeaker_Unified.py`**: 듀얼 스피커 지원 통합 스크립트.
- 각종 보이스별 전용 생성기 (`클래식언니`, `강지영3`, `슬슬이` 등)

---
*모든 .command 파일은 이동된 scripts 폴더 내의 파이썬 파일을 참조하도록 경로가 자동 수정되었습니다.*
