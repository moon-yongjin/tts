# 📜 쇼츠 감성 서사 제작 바이블 (Narrative Guidelines)

사용자님이 정의하신 **실화 기반 1인칭 감성 나레이션 쇼츠**의 제작 로직과 규칙 표준 가이드입니다.

## 뼈대 단계별 5단계 공식 (5-Step Skeleton)

1.  **[Hook] 오프닝 훅**
    *   `[장소/상황]` + `[예상 밖의 행동/결과]` ➡ `[인생이 바뀌었다 / 달라졌다]`
    *   첫 문장에서 끝까지 볼지 여부가 결정됨. **클로징에서 이 첫 줄을 수미상관으로 다시 받아줄 것.**
2.  **[선행] 작고 즉흥적인 행동**
    *   스케일이 작을수록 현실성 높고 공감됨.
3.  **[손해] 불이익 필수**
    *   손해가 없으면 극적 감동이나 클라이맥스 카타르시스가 약해짐.
4.  **[반전] 뜻밖의 연결**
    *   반전은 편당 1~2개가 적당함 (신분, 관계, 기억, 시간, 이유, 타이밍 반전).
5.  **[여운] 감정의 마무리**
    *   설명이나 교훈 없이 감정적인 이미지/한마디로 마무리.

## 🎭 3. 조회수를 터뜨리는 8가지 서사 공식

| 공식명 | 흐름 (Flow) | 예시 훅 (Opening Hook) |
| :--- | :--- | :--- |
| **공식1. 오해** | 나쁜 사람인 줄 알았다 ➡ 사연 발견 ➡ **내가 틀렸다** | "진상 손님인 줄 알았어요. 매일 반만 먹고 가던 이유를 나중에 알았어요." |
| **공식2. 버팀** | 아무도 몰라줬다 ➡ 혼자 견뎠다 ➡ **딱 한 사람이 알아봤다** | "3년을 야근했지만 칭찬 한 번 없었어요. 그러다 퇴사 날 경비 아저씨가 말을 걸었어요." |
| **공식3. 포기직전** | 다 접으려 했다 ➡ 우연한 말 한마디 ➡ **한 번 더 해보기** | "사직서를 출력했어요. 그날 점심 식당 아주머니의 한 마디 때문에 찢었습니다." |
| **공식4. 마지막 사람** | 모두가 떠났다 ➡ 단 한 명이 남았다 ➡ **그 사람이 전부였다** | "회사 망하고 아무도 연락 안 와요. 딱 한 명 빼고요. 근데 제일 의외였어요." |
| **공식5. 역할 교체** | 내가 도와줬다 ➡ 시간이 지나 내가 힘듦 ➡ **그때 그 사람이 등장** | "2년 전 밥을 사줬어요. 잊고 있었는데, 제가 제일 힘들 때 그 사람이 나타났어요." |
| **공식6. 침묵** | 말을 못 했다 ➡ 타이밍 놓침 ➡ 너무 늦음 ➡ **그래도 마지막에 전함** | "고맙다는 말을 10년째 못 했어요. 그러다 오늘 부고 문자가 왔습니다." |
| **공식7. 민낯** | 잘나가는 척했다 ➡ 들켰다 ➡ 치욕감 ➡ **오히려 그게 연결이 됨** | "편의점에서 카드가 거절됐어요. 뒤에 줄이 길었는데, 뒷사람의 행동이 생각나요." |
| **공식8. 은혜** | 작은 선행 ➡ 잊어버림 ➡ **뜻밖의 거대한 보상/화답으로 회귀** | - |

## ✍️ 4. 절대 금지 규칙 (The Forbidden 5)

*   [✕] **선행의 오버 스케일**: 예산이 너무 크거나 영웅적인 행동은 비현실적임.
*   [✕] **지나치게 빠른 보상**: 동화책이 되어버리므로 위기를 충분히 거칠 것.
*   [✕] **감정 설명 금지**: "감동받았다", "슬펐다" 등은 시청자의 감정이입을 차단함.
*   [✕] **교훈 전달 금지**: "착하게 삽시다" 등의 훈계는 이탈을 유발함.
*   [✕] **지나친 극적 연출**: 일일드라마(막장) 톤을 피하고 '일상 연장선'의 담백함을 유지할 것.

## 💬 5. 댓글 유도 훅 배치 (Comment Trigger)

*   **초반**: 공감 유도 ➡ *"여러분이라면 어떻게 하셨을까요?"* (이탈 방지)
*   **중반**: 논란 유도 ➡ *"이런 경험 있으세요? 어떤 게 맞는 것 같나요?"* (댓글 수집)
*   **후반(피날레) - 2부 유도 🌟**: 절벽 엔딩(Cliffhanger) 후 연쇄 시청 및 댓글 증폭 유도
    *   **위치**: 스크립트 맨 아래 '여러분의 생각은 어떠신가요?' 질문 **바로 밑** 고정
    *   **표준 템플릿**: `2부 예고: [다음 화의 반전/궁금한 주제]가 궁금하시면 댓글에 '2부'라고 적어주세요.`
    *   *효과*: 특정 단어("2부")를 반복하게 만들어 유저 댓글 참여율과 피드 알고리즘 부스팅 극대화
